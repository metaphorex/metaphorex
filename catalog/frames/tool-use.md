The relationship between a human agent and an instrument employed to
accomplish work. Tool use involves selection, skill acquisition, ongoing
maintenance, and the gradual extension of capability that comes with
mastery. As a target domain, it is structured by metaphors that import
social relationships (companionship, servitude, partnership) onto what
is fundamentally a one-directional dependency between user and artifact.
