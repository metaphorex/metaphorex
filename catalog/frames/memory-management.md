The allocation, use, and reclamation of computer memory. Programs
request regions of memory, store data in them, navigate to them via
addresses or references, and release them when finished. Memory
management is a rich target domain for spatial and embodied metaphors:
memory has "locations," pointers "point to" addresses, data "lives"
in memory, and unused memory is "freed." The domain's characteristic
failure modes -- leaks, dangling references, overflows -- each carry
their own metaphorical freight.
