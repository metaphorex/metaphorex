The subsystem responsible for parceling out, tracking, and reclaiming
a program's working memory. Memory management is where computing meets
its most fundamental physical constraint -- finite storage -- and the
vocabulary reflects it: allocation, regions, boundaries, overflow. The
domain is a natural target for spatial and container metaphors because
memory genuinely is a bounded resource divided into addressable locations.
