The encoding and manipulation of written language in computer systems.
Text must be represented as sequences of numeric values (characters),
stored in memory with defined boundaries, and interpreted according to
encoding rules. The domain spans character sets, string types, encoding
schemes, and the operations that search, concatenate, and transform
textual data. Computing borrowed its text vocabulary from typography,
bookbinding, and the physical handling of written documents.
