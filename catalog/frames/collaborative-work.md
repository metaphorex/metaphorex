Work done by multiple people toward a shared goal. Encompasses
meetings, code reviews, pair programming, design sessions, and any
context where individual contributions must be integrated. Fertile
target domain for metaphors because collaborative work is inherently
social — power dynamics, communication breakdowns, and coordination
costs are always present.
